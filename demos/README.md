# Demo Gallery

This directory contains demonstrations of holograms and holographic effects.

## Contents

### Generated Simulations
- `fourier_hologram_demo.png` - Example Fourier hologram generation
- `weekly-hologram-*.gif` - Auto-generated weekly hologram animations

### Real Hologram Photos
Add photos of physical holograms you've created here. Recommended format:
- High resolution (at least 1920x1080)
- Multiple viewing angles
- GIF animations showing parallax (if possible)

### Simulation Demos
Add rendered animations of computer-generated holograms:
- Rotating 3D objects
- Interactive hologram effects
- Algorithm comparisons

## Contributing Demos

To contribute a demo:

1. **Real Hologram Photos**
   - Take high-quality photos
   - Show different viewing angles
   - Include brief description in filename or separate text file

2. **Simulation Animations**
   - Use GIF format for web compatibility
   - Keep file sizes reasonable (< 10MB recommended)
   - Include source code reference if applicable

3. **Video Demos**
   - Use MP4 format (H.264 codec)
   - Keep under 100MB if possible
   - Include description and credits
