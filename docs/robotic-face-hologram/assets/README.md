# 3D Head Model

`head.glb` is the **LeePerrySmith** scanned head bust from the [Three.js examples collection](https://threejs.org/examples/models/gltf/LeePerrySmith/LeePerrySmith.glb).

Used under the same terms as the Three.js project (MIT license). The scan is a well-known photorealistic human head mesh used in WebGL demos.

If the local file is missing, the demo falls back to loading from `threejs.org` at runtime.
